# Sudoku Cpp OOP Project

A fully object-oriented Sudoku game built using **C++** and **Qt**.

---

##  Features (Not finished yet) 
- ✔️ Playable Sudoku game with Qt GUI   
- ❌ Difficulty modes (Easy / Medium / Hard)  
- ✔️ Automatic validation  
- ✔️ Automatic Solve 
- ❌ Notes ( Pencil)
- ❌ Best Run
