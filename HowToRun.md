# How to run

## Requirements
- Qt Creator (with CMake and a C++ compiler installed)
- This repository  
    ```console
    > git clone https://github.com/Anish-Pun/Cpp-Sudoku-OOP.git
    ```

---

## Steps

### **✔️ Important — Step 1 & 2**
1. Go to the directory where you cloned the repository.  
2. Open the folder named **`Sudoku`** and **delete the folder named `build`** (this avoids CMake path/drive conflicts).

---

### Continue:
3. Open **Qt Creator**.  
4. Click **Open Project** (top-right or File → Open File or Project).  
5. Navigate to the folder where you cloned the repo.  
6. Select the file named **`CMakeLists.txt`** inside the `Sudoku` directory and open it.  
7. Qt Creator will ask you to *Configure Project*. click **Configure**.  
8. Click **Build** (`hammer icon` bottom left) to compile.  
9. Click **Run** (`green play button` bottom left under `Debug(imported)`) to start the Sudoku application.
